Detailed Description

ESP32-based Temperature Monitoring and Data Logging System.

This code connects to a Wi-Fi network, reads temperature data from a DS18B20 sensor, logs the data to an SD card, and provides a web interface with WebSocket support. It also allows for deep sleep mode and button interrupt to wake up the device.
